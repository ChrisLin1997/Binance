# Binance

## Description

a simple virtual cryptocurrency exchange

https://chris-binance.herokuapp.com/

## Tech Stack

- Frontend
  - `React`
  - `Redux`
  - `TypeScript`
  - `Style-Components`
  - `Vite`
- Backend
  - `Node.js`
  - `Express`
  - `Mysql`

## Start Up
- frontend 
  - `yarn install` or `npm install`
  - `yarn dev` or `npm dev`
- backend
  -  set database configs
  - `yarn install` or `npm install`
  - `yarn start` or `npm start`
